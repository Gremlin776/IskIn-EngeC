Логика psd_comparator.py:
1. Парсинг строк ведомости и сметы:
   "Замена труб 32 мм - 10 м" →
   {description: "Замена труб 32 мм", volume: 10.0, unit: "м"}

2. TF-IDF векторизация описаний (русский язык)
   from sklearn.feature_extraction.text import TfidfVectorizer

3. Cosine similarity для поиска совпадений:
   from sklearn.metrics.pairwise import cosine_similarity
   threshold = 0.65

4. Сравнение объёмов и расчёт расхождения:
   deviation = abs(volume_fact - volume_plan) / volume_plan * 100

5. Итоговый отчёт:
{
  "matches": [
    {
      "work_item": "Замена труб 32 мм",
      "psd_item": "Прокладка трубопровода Ø32",
      "similarity": 0.87,
      "volume_fact": 10.0,
      "volume_plan": 15.0,
      "deviation_pct": 33.3,
      "status": "РАСХОЖДЕНИЕ"
    }
  ],
  "total_items": 10,
  "matched": 8,
  "deviations_found": 3
}
